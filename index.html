Artist: Aesthetically Controllable Text-Driven Stylization without Training

Ruixiang Jiang, Changwen Chen*
The Hong Kong Polytechnic University, *Corresponding Author
Paper arXiv Code Data

TL;DR: Artist stylizes images based on text prompts, preserving the original content while producing high aesthetic quality results. No finetuning, no ControlNets, it just works with your pretrained StableDiffusion model.

Method overview

Artist disentangle the content and style generation into two separate denoising processes with shared information, whose features are used to control the main stylization branch.

Aesthetic Control

Artist allows the user to easily control the abstractness of the stylization, which includes but is not limited to stroke, shape, and color. results.
